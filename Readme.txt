Запуск из исходников выполняется с помощью команды python.exe main.py.
Сборка в исполняемый файл выполняется с помощью пакета pyinstaller.
Результат работы pyinstaller находится в папке dist.
Обратите внимание, что файлы config.json и config_plain.json обязаны находиться в той же директории, что и .exe (либо .py, в случае запуска из исходников) файл.
